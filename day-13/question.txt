Q. Install IP tables nad configure them to allow only LBR host on all the app servers.
Ans: 
    1. `sudo yum install iptables-services -y`
    2. `sudo iptables -A INPUT -p tcp --dport 3000 -s LBR_IP -j ACCEPT`  - to allow only request from LBR_IP
        `sudo iptables -A INPUT -p tcp --dport 3000 -j DROP` - to block all other requests

    3. `sudo service iptables save`
        `sudo systemctl enable iptables`

    4. `iptables -L`
